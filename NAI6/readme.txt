Cel projektu:
Aplikacja, która wykorzystuje detekcję twarzy i oczu do zatrzymania/wznawiania odtwarzania reklamy wideo.
Program monitoruje, czy użytkownik patrzy na ekran (na podstawie oczu), a gdy wykryje, że oczy są zamknięte
przez określoną liczbę klatek, zatrzymuje reklamę.

Aplikacja ma "nasladowac" srodowisko z odcinka Fifteen Million Merits z serialu Black Mirror (S1 E2)
https://youtu.be/oENhTCDr1UY?si=a8CjrsTuRRi3i6_Q&t=19

Jeśli posiadamy zainstalowanego pythona wystarczy zainstalować biblioteki użyte w projekcie za pomocą:
pip install opencv-python
pip install dlib
pip install imutils

Do dlib potrzebna jest instalacja CMake i dodatnie CMake/bin do zmiennych środowiskowych!!!!
